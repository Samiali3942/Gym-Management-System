
# 🏋️‍♂️ Gym Management System

A simple and efficient Gym Management System that allows gym owners or administrators to manage daily operations including:

✅ Member registrations

📅 Class schedules

👨‍🏫 Trainer assignments

💸 Payment tracking

🕒 Attendance logging





## 🛠️ Tech Stack



Frontend: HTML/CSS

Backend: Python

Database: MySQL
